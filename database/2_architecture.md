# 7. ANSI/SPARC 아키텍쳐와 데이터 독립성

## ANSI/SPARC 아키텍쳐

- 물리적, 개념적, 외부 단계 3단계로 이루어짐
  - 외부 단계(external level): 각 사용자의 뷰
  - 개념 단계(conceptual level): 사용자 공동체의 뷰
  - 내부 단계(internal level): 물리적 또는 저장 뷰
